# Task-Escalation-Monitoring-System

This project is a role-based Task Escalation and Monitoring System built using Streamlit and
Google Sheets integration. It enables task assignment, tracking, escalation management, and
real-time analytics for enterprise-level workflow monitoring.
In large organizations, tasks often get delayed or overlooked without a proper follow-up mechanism. Managers struggle to track which tasks are overdue, which employees need escalation, and how departments are performing overall. This system solves that by automating the entire task monitoring and escalation pipeline — from assignment to real-time analytics.
